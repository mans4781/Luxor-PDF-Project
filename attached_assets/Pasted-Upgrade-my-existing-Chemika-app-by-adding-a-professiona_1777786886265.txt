- Copy for PageMaker uses PNG fallback.
- Print opens browser print for current equation/canvas.

- Close Equation closes/clears current equation area.

## Menu component behavior

Menu bar:
- Click menu title to open dropdown.
- Hover to switch between open menus.
- Click outside closes menu.
- Esc closes menu.
- Arrow keys can navigate menu if possible.
- Enter activates selected item if keyboard navigation is implemented.

Menu item data structure:

{
  id: "file-new",
  menu: "File",
  label: "New Equation",
  shortcut: "Ctrl+N",
  action: "newEquation",
  enabled: true
}

Submenu data structure:

{
  id: "preference-theme",
  label: "Theme",
  submenu: [
    { label: "Light", action: "setThemeLight" },
    { label: "Dark", action: "setThemeDark" },
    { label: "System", action: "setThemeSystem" }
  ]
}

## Central command system

Create a central command dispatcher:

executeChemikaCommand(commandId, context)

All menu items, toolbar buttons, and keyboard shortcuts should call the same command system.

Examples:
executeChemikaCommand("insertSubscript")
executeChemikaCommand("insertSuperscript")
executeChemikaCommand("copyForWord")
executeChemikaCommand("openCompoundLibrary")
executeChemikaCommand("toggleAutoChemistryFormat")
executeChemikaCommand("openKeyboardShortcuts")

This avoids duplicate logic.
