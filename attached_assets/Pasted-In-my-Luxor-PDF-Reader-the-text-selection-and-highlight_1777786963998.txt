Main behavior:
1. When user selects text, collect all selection rectangles from the PDF text layer.
2. Merge rectangles line-by-line.
3. For each selected line:

- Remove tiny/empty rectangles.

- Merge only rectangles from the same line.
- Do not merge rectangles from different columns or different questions if they are far apart horizontally.
- Because my PDF has two columns, the logic must respect columns.
- Do not create one long rectangle that crosses from the left column to the right column.
- If selected rects are on the same Y level but far apart horizontally, treat them as separate line groups.
- Add a horizontal gap threshold to prevent merging across columns.

Line merging logic:
- For each rect, calculate:
  - centerY = y + height / 2
  - left = x
  - right = x + width
- Group rects into the same line if:
  - their centerY values are close within a tolerance, for example 3–6 px depending on zoom
  - and they are on the same page
  - and the horizontal gap between neighbouring rects is not too large
- For each line group:
  - mergedX = minimum x
  - mergedY = minimum y
  - mergedRight = maximum x + width
  - mergedBottom = maximum y + height
  - mergedWidth = mergedRight - mergedX
  - mergedHeight = mergedBottom - mergedY
- Use a consistent line height for the merged rectangle.
- Slightly adjust top and bottom padding so the overlay looks clean, but do not cover the line above or below.

Two-column PDF handling:
- If two selected rectangles have similar Y positions but are separated by a large horizontal gap, do not merge them.
- Example:
  - Left column question line and right column question line may have same Y value.
  - They must remain separate selection rectangles.
- Add `maxHorizontalGap` logic:
  - normal spaces between words should be merged
  - large gaps between columns should not be merged
- Suggested:
  - Merge if gap is less than 25–40 px at current zoom, or dynamically based on average character/word gap.
  - Do not merge if gap is larger than a column gap threshold.
