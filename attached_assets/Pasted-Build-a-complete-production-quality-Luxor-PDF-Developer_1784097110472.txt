## 14. Referral Program

Create a complete referral-management section.

Referral rule:

* Every registered user receives a unique referral link.
* When a new customer purchases a paid subscription through that link, the new customer receives a 10% discount.
* The referring user receives two additional months of subscription validity.
* Rewards should only be issued after successful payment.
* Prevent users from referring themselves.
* Prevent duplicate reward claims for the same purchase.
* Record all referral activity in an audit log.

Display a prominent rule badge:

**Friend gets 10% OFF • Referrer gets +2 months validity**

Create a visual three-step flow:

1. Existing user shares unique referral link.
2. New user purchases a subscription and receives 10% discount.
